# aurora-login-portal
# Overview
+ A modern, responsive login interface featuring glassmorphism design, animated mesh gradients, and smooth user interactions.
## Features include:
+ Glassmorphism UI: Frosted glass effect using backdrop-filter.
+ Animated Background: A fluid, breathing mesh gradient.
+ Typography: Use of Sans Serif Display and Montserrat.
+ Interactivity: Smooth hover effects and button transitions.
